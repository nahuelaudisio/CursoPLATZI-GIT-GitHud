Gitblog Tu camino para ser un maestro en git

¿Cómo trabajar con Git y Github sin morir en el intento?

En este post trataremos los temas de git y github para el control de versiones local y remoto, crearemos ramas, resolveremos conflictos, subiremos los archivos a un repositorio remoto, aprenderemos a trabajar en equipo y mucho más. Puedes consultar el archivo README.md que tiene todos los conceptos sobre Git y Github cuando lo amerites.

Git y Github son tremendamente útiles a la hora de trabajar con control de versiones, trabajos colaborativos y para mostrar al mundo tu conocimiento. Es de mucha ayuda en tu carrera como desarrollador tener un Github saludable donde los demás puedan chequear tus habilidades.
